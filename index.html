Sobre

Sou desenvolvedor Full Stack especializado em JavaScript e TypeScript, com foco em aplicações web para gestão comercial e industrial. Trabalho com Node.js no backend, React no frontend e SQL Server como principal banco de dados. Tenho formação em desenvolvimento de APIs (JS Expert) e experiência com Python em projetos de IA. Atualmente, busco evoluir em áreas como Inteligência Artificial, Machine Learning, criação de dashboards interativos em React e otimização de performance no backend com Node.

Backend

Atuo na integração entre o client-side e o banco de dados por meio de APIs bem estruturadas. Dou ênfase à performance das aplicações, explorando recursos como programação assíncrona, execução paralela e processamento sob demanda no Node.js. Adoto TDD para garantir confiabilidade, seguindo princípios de Clean Code e mantendo uma arquitetura organizada e coerente.

Frontend

Desenvolvo interfaces modernas e profissionais, com atenção à responsividade, performance e interatividade. Utilizo componentes reutilizáveis e estilizações consistentes, sempre alinhadas às boas práticas de UI/UX. A estrutura do projeto é bem definida, com organização clara de pastas, o que facilita a manutenção e garante escalabilidade, mantendo a estética e a usabilidade como prioridades.

IA

Atuo com Inteligência Artificial, focando em Machine Learning e integração com modelos de mercado. Tenho experiência com JavaScript, Python, NLP, embeddings, previsão estatística e análise de imagem em tempo real. Utilizo essas tecnologias para enriquecer APIs, melhorar previsões e criar experiências mais inteligentes e personalizadas.
